Forca

O projeto da forca visa fazer um jogo popular com elementos de programação frontend usando html, css e javascript. O jogo consiste em acertar uma palavra adivinhando sua letras e quando errar uma parte de um boneco palito é acrescentada e depois de 6 erros o jogo acaba.
